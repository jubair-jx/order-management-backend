# Order Management Backend(Mongoose TypeScript, CRUD Operations)

This project is an Order Management backend API built using Mongoose, Express, and TypeScript.

## Prerequisites

Before you begin, ensure you have the following installed:

- Node.js
- npm or yarn
- MongoDB

## Getting Started

### Installation

order-management-backend/
├── src/
│ ├── app/
│ ├── config/
│ ├── modules/
│ ├── app.ts
│ └── server.ts
├── .env
├── .gitignore
├── package.json
├── tsconfig.json
└── README.md

### Run Project

`Step:1` For development purpose

```typescript
 npm run start:dev
```

`Step:2` For Production

```typescript
yarn build
```
